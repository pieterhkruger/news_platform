THE DAILY INDABA - DEMO CREDENTIALS
==================================
All accounts are created automatically after `python manage.py migrate` on a
fresh database. The same records can also be created manually with the seed
command shown below.
Every account shares the same password. Passwords are intentionally simple -
do not use in production.

Automatic setup:  python manage.py migrate
Seed command:     python manage.py seed_demo_news
Custom password:  python manage.py seed_demo_news --password <yourpassword>

NOTES
-----
- To rebuild a wiped database:      python manage.py migrate
- To re-seed a migrated database:   python manage.py seed_demo_news
- To force-update existing records: python manage.py seed_demo_news --update-existing
- Seed data lives in:               daily_indaba/seed_data/default_news_data.json
- Article HTML bodies live in:      daily_indaba/seed_data/articles/
- Seed images live in:              daily_indaba/seed_data/images/
